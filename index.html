What is it?

Longitude is a set of calculations for creating mobile first, responsive and semantic web sites. It is written purly in SCSS and is meant to be as unobtrusive as possible, meaning that you shouldn't have to adapt your semantics to the CSS.

Since Longitude only consits of calculations, it doesnt add any overhead to your CSS. No extra stylesheets to include in your final CSS, nothing.

What is it not?

Longitude is not a set of classes, it is a set of calculations. This means that Longitude itself is not a grid. You have to create the grid!

Fear not! Longitude makes it a breeze. It is acctually really handy to be able to taylor the grid exclusivly for the web site you are creating!

Longitude is not a collection of UI styles, it just helpes you with the responsive layout, thats all. So if you want to combine it with something else, go ahead! Or if you want to roll your own, please feel free. Longitude will stay out of your way.

Example

Here are some examples of how you can make your own grid. Longitude uses columns as units. Look at the following SCSS: 

/* Breakpoints in columns */
@include longitude($breakpoints: 4 8) {
  section {
    float: left;
    /* 2 per row when narrow layout, and 4 when wide */
    width: columns(2);
    /* this gives us some space between the sections */
    @include gutters;
  }
}

It translates to CSS like this: 

/* Breakpoints in columns */
@media (max-width: 320px) {
  section {
    float: left;
    /* 2 per row when narrow layout, and 4 when wide */
    width: 43.75%;
    /* this gives us some space between the sections */
    margin-left: 3.125%;
    margin-right: 3.125%; } }
@media (min-width: 321px) and (max-width: 640px) {
  section {
    float: left;
    /* 2 per row when narrow layout, and 4 when wide */
    width: 21.875%;
    /* this gives us some space between the sections */
    margin-left: 1.563%;
    margin-right: 1.563%; } }
@media (min-width: 641px) {
  section {
    float: left;
    /* 2 per row when narrow layout, and 4 when wide */
    width: 140px;
    /* this gives us some space between the sections */
    margin-left: 10px;
    margin-right: 10px; } }
